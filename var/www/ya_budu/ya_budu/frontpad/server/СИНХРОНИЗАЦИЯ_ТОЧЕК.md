# Инструкция по синхронизации заказов с несколькими точками

✅ На стороне Frontpad всё уже настроено и готово.

## Что нужно сделать на основном сайте (3 простых шага):

### Шаг 1. При получении районов доставки сохраняйте point_id
Когда вы получаете список районов с Frontpad:
```javascript
const response = await fetch('https://ваш-frontpad.ru/api/site/delivery-zones', {
  headers: { 'X-Sync-Token': 'ВАШ_ТОКЕН' }
});
const zones = await response.json();

// ✅ Каждый объект зоны уже имеет поле point_id!
// Пример ответа:
// [
//   {
//     id: 1,
//     name: "Центр",
//     min_order_amount: 1000,
//     delivery_price: 150,
//     point_id: 1  // ✅ этот район обслуживает точка 1
//   },
//   {
//     id: 2,
//     name: "Юг",
//     min_order_amount: 1000,
//     delivery_price: 200,
//     point_id: 2  // ✅ этот район обслуживает точка 2
//   }
// ]
```

### Шаг 2. Когда пользователь выбирает район - запомните его point_id
```javascript
// Когда пользователь выбрал район доставки:
const selectedZone = zones.find(z => z.id === userSelectedZoneId);
const orderPointId = selectedZone.point_id; // ✅ это и есть та точка которая получит заказ
```

### Шаг 3. При синхронизации заказа добавьте ОДНУ СТРОЧКУ
```javascript
await fetch('https://ваш-frontpad.ru/api/site/orders/sync', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-Sync-Token': 'ВАШ_ТОКЕН'
  },
  body: JSON.stringify({
    // ... все ваши старые поля заказа ...
    order_id: 12345,
    guest_name: "Иван",
    guest_phone: "79001234567",
    items: [...],
    total_amount: 1500,
    
    // ✅ ДОБАВЬТЕ ЭТУ ОДНУ СТРОЧКУ!
    point_id: orderPointId
  })
});
```

✅ Готово! Больше ничего менять не нужно.

## Как это будет работать:
1. Пользователь выбирает район "Центр" → point_id = 1 → заказ попадёт ТОЛЬКО в точку 1
2. Пользователь выбирает район "Юг" → point_id = 2 → заказ попадёт ТОЛЬКО в точку 2
3. Админ точки 1 видит только свои заказы
4. Админ точки 2 видит только свои заказы
5. Автопечать, уведомления, всё остальное работает как раньше

## Что ещё нужно настроить в Frontpad:
1. Зайдите в админку Frontpad под admin
2. Перейдите в Районы доставки
3. Для каждого района укажите какая точка его обслуживает
4. Сохраните

✅ Всё! Система полностью готова к работе с двумя независимыми точками.
