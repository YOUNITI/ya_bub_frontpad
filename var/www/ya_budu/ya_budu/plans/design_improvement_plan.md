# План улучшения дизайна сайта "Я Буду!"

## Анализ текущего дизайна

После просмотра компонентов Hero.jsx, Header.jsx, Footer.jsx, ProductCard.jsx и globals.css, сайт имеет стандартный дизайн с использованием базовых цветов: желтый (#fbbf24), оранжевый (#f97316), серый и белый. Компоненты используют простые формы, минимальные анимации (fadeIn, hover scale на изображениях), и отсутствует тематический дизайн, отражающий специфику азиатской кухни (роллы, пицца). Это делает сайт "обычным" из-за:
- Стандартной цветовой палитры без уникальности.
- Отсутствия декоративных элементов, связанных с тематикой.
- Минимальных анимаций и эффектов.

## Предлагаемое дизайнерское решение

Для улучшения внедрить тематический дизайн азиатской кухни:
- **Цветовая схема**: Красный (#DC143C), черный (#000000), золотой (#FFD700), темно-красный (#8B0000).
- **Градиентные фоны**: Градиенты от красного к золотому для создания динамики.
- **Современные шрифты**: Добавить шрифт с азиатским оттенком, например, "Noto Sans JP" или кастомный шрифт, имитирующий каллиграфию.
- **Анимации**: Ввести floating elements (плавающие элементы), parallax эффекты, сложные hover анимации (например, rotate, glow).
- **Уникальные элементы**: Декоративные паттерны (бамбук, иероглифы, драконы), фонарики, рамки в азиатском стиле.

Это добавит уникальность, сделает сайт запоминающимся и соответствующим тематике.

## Детальный план изменений

### 1. Обновление src/styles/globals.css
- Добавить CSS-переменные для новых цветов.
- Импортировать новые шрифты.
- Добавить новые анимации и эффекты.

Примеры CSS классов:
```css
:root {
  --asian-red: #DC143C;
  --asian-gold: #FFD700;
  --asian-black: #000000;
  --asian-dark-red: #8B0000;
}

@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;700&display=swap');

body {
  font-family: 'Noto Sans JP', 'Montserrat', sans-serif;
}

.animate-floating {
  animation: floating 3s ease-in-out infinite;
}

@keyframes floating {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}

.animate-glow {
  animation: glow 2s ease-in-out infinite alternate;
}

@keyframes glow {
  from { box-shadow: 0 0 5px var(--asian-gold); }
  to { box-shadow: 0 0 20px var(--asian-gold), 0 0 30px var(--asian-gold); }
}

.asian-pattern {
  background-image: url('data:image/svg+xml,<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100"><text fill="var(--asian-gold)" font-size="50" y="50%">竹</text></svg>');
  background-repeat: repeat;
  opacity: 0.1;
}
```
